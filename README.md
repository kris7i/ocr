<h1 align="center"> ocr </h1>

<p align="center"> 基于百度API的图片识别.</p>


## 安装

```shell
$ composer require expnull/ocr -vvv
```

## 配置

在使用本扩展之前，你需要去 [百度AI开放平台](http://ai.baidu.com/) 注册账号，然后创建应用，获取应用的 API Key
